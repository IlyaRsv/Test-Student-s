**ОБНОВИТЬ репозиторий

Откройте терминал и перейдите в директорию вашего локального репозитория.
Выполните следующую команду, чтобы увидеть текущие настройки удалённого репозитория:
git remote -v

Обновите URL удалённого репозитория, используя команду git remote set-url, например:
git remote set-url origin новый-URL-репозитория
Замените новый-URL-репозитория на актуальный URL вашего репозитория на GitHub.

**УДАЛИТЬ последнее изменение в репозитории и редакторе кода
Команда `git reset --hard HEAD^` откатывает изменения только в вашем рабочем каталоге (редакторе кода), но не в репозитории. Для того чтобы отменить последний коммит и вернуться к предыдущему состоянию репозитория, вам также нужно выполнить следующую команду:

git push --force

Эта команда отправит изменения в удалённый репозиторий, заменяя последний коммит предыдущим состоянием. Пожалуйста, будьте осторожны, так как `--force` форсирует обновление удалённой ветки и может повлиять на других разработчиков, работающих с этим репозиторием.
